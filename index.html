Manual Installation Instructions

  1. Visit our GitHub repository and download the source code zip file.
  2. Open Chrome browser and navigate to chrome://extensions/
  3. Enable Developer mode, then click on Load unpacked and select the unzipped folder.
  4. The extension is now installed and ready for use.
